# MeowCadSystem

Учебный CAD-редактор на JavaFX для построения и редактирования 2D-примитивов.

## Возможности

- Построение отрезков, окружностей, дуг, прямоугольников, эллипсов, многоугольников и сплайнов
- Несколько способов создания примитивов: по точкам, по размерам, по центру и параметрам
- Ручной ввод длины, радиуса, диаметра, ширины, высоты и других параметров
- Редактирование через контрольные точки
- Привязки: конец, середина, центр, пересечение, перпендикуляр, касательная
- Стили линий по ЕСКД/ГОСТ и пользовательские стили
- Поддержка слоёв
- Импорт и экспорт DXF
- Масштабирование, панорамирование и поворот вида

## Технологии

- Java 17
- JavaFX 17
- Maven Wrapper

## Запуск

Из корня проекта:

```bash
./mvnw javafx:run
```

Для Windows:

```powershell
mvnw.cmd javafx:run
```

## Сборка

Собрать проект:

```bash
./mvnw clean compile
```

Собрать fat JAR:

```bash
./mvnw clean package
```

После сборки итоговый JAR находится в `target/`.

## Структура проекта

- `src/main/java/org/example/cadsystem` - точка входа приложения
- `src/main/java/org/example/model` - модели примитивов, стили, слои, привязки
- `src/main/java/org/example/view` - интерфейс, тулбары, отрисовка, диалоги
- `src/main/java/org/example/controller` - обработка ввода и взаимодействие между UI и моделью
- `src/main/java/org/example/export` - импорт и экспорт DXF
